# Flower of Life Neural Networks

Обучаемые нейросети на геометрических решётках, порождённых принципом
«Цветка жизни»: узлы — окружности (в 3D — шары), рёбра — их пересечения,
сигнал распространяется волнами message passing с общими весами.

Серия из 12 экспериментов на чистом numpy (ручной backprop, проверенный
численным градиентом). Полное описание методики, результатов и выводов —
в [FLOWER_OF_LIFE_REPORT.md](FLOWER_OF_LIFE_REPORT.md).

## Ключевые результаты

- Структура «как у Цветка жизни» обучаема в 2D и 3D; геометрия даёт
  +18 пунктов точности против случайного графа при равных параметрах.
- Особенна не шестиугольность (квадратная решётка работает так же), а
  регулярность + локальность + общие веса.
- 203 параметра решётки бьют MLP с 8 387 параметрами; при 50 примерах
  решётка обходит MLP, обученный на 900.
- Локальное правило переносится на решётки другого размера (zero-shot до
  469 узлов) и частично из 2D-сечения в 3D-объём.
- В 3D по правилу Цветка жизни у шара 18 соседей (не 12, как у касающихся
  шаров в FCC) — 6 в плоскости + по 6 сверху и снизу.

## Файлы

| Скрипт | Что показывает |
|---|---|
| `flower_of_life_net.py` | 2D: базовая обучаемость (96.5%) |
| `flower_of_life_net_3d.py` | 3D FCC-упаковка; срез (111) = плоский Цветок |
| `flower_of_life_grow.py` | Рост с одной окружности, веса наследуются |
| `flower_of_life_dense.py` | 6 против 12 соседей, пере-сглаживание |
| `flower_of_life_baselines.py` | Вклад геометрии против логрег/MLP/случайного графа |
| `flower_of_life_square.py` | Гекс не особен: квадрат работает так же |
| `flower_of_life_transfer.py` | Zero-shot перенос на решётки до 469 узлов |
| `flower_of_life_18.py` | 18 соседей (правило Цветка) против 12 (FCC) |
| `flower_of_life_damage.py` | Живучесть: плавная деградация при повреждениях |
| `flower_of_life_data.py` | Эффективность по данным |
| `flower_of_life_2d3d.py` | Перенос правила из сечения в объём |
| `flower_of_life_regress.py` | Локализация центра узора вниманием |

У каждого скрипта параметры запуска описаны в шапке. Рядом с каждым —
одноимённый `.png` с результатами.

## Запуск

```bash
pip install numpy matplotlib scipy
python flower_of_life_net.py
```

Веса и промежуточные состояния (`*.npz`) не хранятся в репозитории
(см. `.gitignore`) — они воспроизводятся запуском скриптов.
